# Calculate Harry Potter Books

Install chai and mocha
```
npm i
```

Run tests
```
npm test
```

Input is an object with the books as keys and quantity as value. Example:
```
{
    var cart = {
            'book1': 3,
            'book2': 3 ,
            'book3': 2,
            'book4': 2,
            'book5': 3
        }
}
```
